# Iris Protocol Documentation

> *The Foundation issues the credit lines; the network executes the reality of the ledger.*

Iris Protocol is a modular leveraged trading stack built around **IrisX** (`IXToken`) — a dual-ledger vault token — on-chain governance with voting escrow, and an elite **15-Chair Foundation** overlay that captures protocol performance fees and maintains tactical veto authority.

## Core Components

- **IXToken (IrisX)** — Rebasing vault share over underlying USDT with a parallel fixed 1:1 ledger for DEX safety
- **VotingEscrow** — Lock IXToken for governance weight (block-number clock)
- **IrisGovernor** — Community proposals routed through a timelock controller
- **The Iris Foundation** (`IRIS-FOUNDATION`) — 15 ERC721 Chairs at `0x00008c80D4cBD653B1D384566d9b23B37d100000`
- **5 Keeper NFTs** — Premium execution keys for liquidate and force-close operations
