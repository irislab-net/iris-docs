# Iris Foundation — Brand Guidelines

## Palette

| Name | Hex | Usage |
|------|-----|-------|
| Cosmic Void Black | `#0d0d12` | Backgrounds, dark mode base |
| Holographic Violet | `#6b4ce6` | Primary accent, links, CTAs |
| Silent Chrome | `#c0c0cc` | Secondary text, borders |
| Radiant Gold | `#d4a017` | Foundation / premium highlights |
| Security Crimson | `#c0392b` | Warnings, liquidation states |

## Typography

- **Headings:** System UI stack, weight 700, tight letter-spacing
- **Body:** System UI stack, weight 400
- **Code:** Monospace, 95% size

## Logo

Primary logo: `static/img/logo.svg` (derived from `logo-dark.svg`)

## Voice & Tone

- Technical precision with institutional authority
- Lore sections may use dramatic narrative; whitepaper sections remain academic
- Always distinguish Foundation fee (5%) from Keeper execution incentives

## NFT Art Direction

Foundation Chair renders use cosmic/institutional aesthetic. All 15 Chairs are visually distinct in art but functionally identical on-chain.
